How it works: pick a tool, a length and a background, tweak the captions, then export. Export this clip = the current tool at the current length. All 4 lengths = 3s/6s/9s/12s of the current tool. Every tool = one clip per tool plus the “all tools” showcase, at the current length — exactly what you’d batch-schedule across TikTok, Instagram and YouTube Shorts. Recording is real-time (a 12s clip takes 12s) and downloads automatically. If your browser exports .webm and a phone app refuses it, upload from a desktop or run it once through any free MP4 converter — most modern Chrome builds already export .mp4 directly.
